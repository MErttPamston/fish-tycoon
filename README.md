# 🐠 Fish Tycoon - Симулятор разведения рыб

## Описание

Fish Tycoon — это казуальная ностальгическая игра-симулятор разведения рыб. Игрок начинает с маленького аквариума и постепенно расширяет его, разводя разные виды рыб и зарабатывая деньги.

## Характеристики

- **8 видов рыб**: От количества золотых на до капитальных робо-рыб
- **Полная анимация**: Монынии частиц, рыбы на анимации
- **Синтезированная музыка**: Lo-Fi чил-хоп беккграунд
- **Система уровней**: 50 уровней на прогрессию
- **Ностальгический дизайн**: Пастельные цвета, нежная упаковка
- **Открытая реализация**: 6500+ строккода

## Куда играть

Натируйте инстрскцию, чтобы наслаждаться симуляцией разведения рыб!

```bash
# На васком сервере
1. Получите репозиторий
cd fish-tycoon

2. Послете index.html в браузер

3. По вроде!
```

## Операция

### Новичка
- Кликните рыбу, чтобы выбрать её
- Нажмите "🛒 Магазин", чтобы покупить новых рыб
- Нажмите "🍖 Кормить", чтобы кормить всех

### Горэвые клавиши
- `ESC` - закрыть меню
- `R` - сбросить гре (вы второго раза?)
- `S` - сохранить статус

## Поддержка и Кодовые Правила

- Код целиком на нетивном JavaScript (HTML5 Canvas)
- Нет внешних зависимостей
- Синтетическая музыка (Web Audio API)
- Поддержка LocalStorage
