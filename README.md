## 🎮 Как играть

Откройте `index.html` в браузере — никакого сервера не нужно!

### Основной цикл
```
Покупай рыб → Корми → Разводи → Продавай → Расширяй аквариум
```

### Горячие клавиши
| Клавиша | Действие |
|---------|----------|
| `1` | Магазин |
| `2` | Рыбы |
| `3` | Апгрейды |
| `4` | Статистика |
| `F` | Покормить всех |
| `S` | Продать всех |
| `H` | Вылечить всех |
| `Click` на рыбу | Управление рыбой |

## ✨ Особенности

- 🎵 **Чиптюн-музыка** — синтезируется прямо в браузере (Web Audio API)
- 🔊 **Звуковые эффекты** — монеты, лечение, размножение, смерть
- 🐠 **Живая анимация** — рыбы плавают, водоросли колышутся, пузырьки поднимаются
- 💾 **Автосохранение** — каждые 30 секунд в LocalStorage
- 📺 **CRT-эффект** — сканлайны, глитч, мерцание как на старом мониторе
- 🏆 **14 достижений** — от первой продажи до миллионера
- ⬆️ **6 видов апгрейдов** — размер аквариума, автокормушка, ускоритель роста и др.
- 🎯 **50 уровней** прогрессии

## 🛠️ Технологии

- **Чистый HTML5 + CSS3 + Vanilla JS** — ноль зависимостей
- **Canvas API** — рендеринг аквариума
- **Web Audio API** — синтез музыки и звуков
- **LocalStorage** — сохранение прогресса
- Шрифты: Press Start 2P, VT323, Orbitron (Google Fonts)
