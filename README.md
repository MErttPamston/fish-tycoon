# 🐠 Fish Tycoon - Симулятор разведения рыб

## Описание

Fish Tycoon — это казуальная ностальгическа�� игра-симулятор разведения рыб. Игрок начинает с маленького аквариума и постепенно расширяет его, разводя разные виды рыб и зарабатывая деньги.

## Характеристики

- **8 видов рыб**: От количества золотых на до капитальных робо-рыб
- **Полная анимация**: Движения частиц, рыбы на анимации
- **Синтезированная музыка**: Lo-Fi чилл-хоп фоновая музыка
- **Система уровней**: 50 уровней на прогрессию
- **Ностальгический дизайн**: Пастельные цвета, нежная упаковка
- **Открытая реализация**: 6500+ строккода

## Где играть

Натирайте инструкцию, чтобы наслаждаться симуляцией разведения рыб!

```bash
# На васком сервере
1. Получите репозиторий
cd fish-tycoon

2. Пошлете index.html в браузер

3. По вроде!
```

## Операция

### Новичка
- Кликните рыбу, чтобы выбрать её
- Нажмите "SHOP Магазин", чтобы купить новых рыб
- Нажмите "FOOD Корм", чтобы кормить всех

### Горячие клавиши
- `ESC` - закрыть меню
- `R` - сбросить игру (вы второго раза?)
- `S` - сохранить статус

## Достижения

- [GOLD] "Первая продажа" - Продайте свою первую рыбу
- [MONEY] "Миллионер" - Заработайте 1,000,000 монет
- [ALL] "Коллекционер" - Откройте все виды рыб
- [STAR] "Легенда" - Основные 50 уровней
- [FISH] "Население" - Овладеть 100 рыбами

## Поддержка и Кодовые Правила

- Код целиком на нетивном JavaScript (HTML5 Canvas)
- Нет внешних зависимостей
- Синтетическая музыка (Web Audio API)
- Поддержка LocalStorage
